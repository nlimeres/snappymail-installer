# ⚡️ snappymail-installer
Unofficial installation script for SnappyMail. Works with the latest version of SnappyMail!

Read more about [SnappyMail](https://snappymail.eu) here. This script is not associated with the official SnappyMail Project.

SnappyMail is an extremely lightweight, fast, and modern webmail client (a fork of RainLoop), designed to offer a seamless experience without the security vulnerabilities of its predecessor.


## Features

- "Zero-Configuration" Installation: Automatically detects the installed PHP version and configures the Nginx socket for you.
- Optimized for performance: Configured to run on Nginx and PHP-FPM, consuming minimal resources on your server.
- Enhanced Security: blocking of access to the data folder,disabling hidden files and a strict management of file and folder permissions.
- Lightweight: It does not include unnecessary dependencies, keeping the Debian system as clean as possible.
- Compatible with PHP 8.x: Full support for the latest versions of PHP and its extensions.
